Polar is currently in BETA as we're preparing to release 1.0 in the next 30-60 days.

We're focused on stability first so if you have any problems please let us know by reaching out to us on Discord or creating an issue on Github.

We have builds for Windows, MacOS, and Linux. Currently none of our builds are signed for Windows and MacOS so you will have to manually approve the builds on each platform. For Linux we provide .deb and tar.gz packages.
